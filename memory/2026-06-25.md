# 2026-06-25 Daily Notes

## Venus OS Pi Display Timeout
- Working Pi: `192.168.1.140`
- SSH login used during repair: `root` / `1973Thing`
- Real timeout setting is `Gui/DisplayOff`, not `Gui2/StartPageTimeout`
- Set via SSH/DBus:
  - `dbus-send --system --print-reply --dest=com.victronenergy.settings /Settings/Gui/DisplayOff com.victronenergy.BusItem.SetValue int32:300`
- The screen blanking target that actually worked was:
  - `/sys/class/backlight/rpi_backlight/bl_power`
- `blank_display_device` should contain:
  - `/sys/class/backlight/rpi_backlight/bl_power`
- The red power LED was turned off via `/data/rc.local` with:
  - `echo none > /sys/class/leds/led1/trigger`
  - `echo 0 > /sys/class/leds/led1/brightness`
- If the Display Off menu item vanishes, switch `Gui/RunningVersion` to `1` and restart `/service/gui`
