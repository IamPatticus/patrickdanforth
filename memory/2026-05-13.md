## OpenClaw Update Saga (2026.5.4 → 2026.5.7)
- Started at ~10:12 CDT, finished at ~14:51 CDT (nearly 5 hours).
- **Root cause:** Missing user D-Bus socket `/run/user/1000/systemd/private` — all `systemctl --user` commands hang indefinitely. Gateway kept respawning (`Restart=always`) before npm could finish installing.
- **What didn't work:** Interactive shell, background `&`, system crontab, OpenClaw cron, nohup, setsid, Python subprocess — all got killed when the gateway stopped/restarted, or D-Bus timed out.
- **What worked:** Patticus ran it directly via SSH terminal: `systemctl --user stop` → `npm install -g openclaw@2026.5.7` → `systemctl --user start`.
- **Result:** 2026.5.7 is live. Gateway healthy.
- **Lesson learned:** For OpenClaw updates on this Pi, always use direct SSH, never an agent-managed approach.
