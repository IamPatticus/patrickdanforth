### Tauri Desktop App Build

- **Bundle:** `headache-app-desktop.zip` uploaded to Google Drive folder `Headache Desktop` (ID: `1IZVYEuPulusgoAhgsgPEz8VQZiNtxfSo`)

- **Problem 4 (window label collision):** `"main"` window persisted after crash. `get_webview_window` approach resolved it
- **Key files:** `src/main.rs`, `src/routes.rs`, `src/db.rs`, `src/models.rs`, `src/frontend.rs`, `frontend/headache.html`
- **Build command:** `npx tauri build` on Windows (requires Rust + Node.js)
- **Output:** `headache-app-desktop/target/release/headache-app-desktop.exe`
