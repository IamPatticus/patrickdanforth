assistant: Quickest way — you've got `fswebcam` already:

```bash
# Basic snap
fswebcam image.jpg

# With the Kiyo (usually /dev/video0)
fswebcam -d /dev/video0 image.jpg

# Skip frames so auto-exposure settles (2 frame skip)
fswebcam -d /dev/video0 -S 2 image.jpg

# With ffmpeg (for more control)
ffmpeg -f v4l2 -i /dev/video0 -frames:v 1 photo.jpg
```

`fswebcam -S 2` is the one I'd use — lets the sensor adjust exposure before grabbing the frame. The first couple frames from a cold start are always blown out.
