# Clawcar Avatar Prompt

Use this prompt with any image generator once credits or a local model are available.

**Aspect ratio:** 1:1  
**Style:** Comic-meets-tech character avatar, bold colors, clean lines, suitable for a small icon.

**Prompt:**

> A tiny robotic stunt car with a cheeky cybernetic lobster mascot riding inside, Reginald-style chaos energy, sarcastic expression, bold colors, compact SunFounder PiCar-X chassis with a camera gimbal, comic-meets-tech illustration style, clean character avatar suitable for a 1:1 icon.

**Negative prompt (optional):**

> photorealistic, blurry, dark background, cluttered, realistic textures, grim, horror.

**Output filename:** `clawcar-avatar.png`  
**Place at:** `avatars/clawcar-avatar.png` in the OpenClaw workspace.
